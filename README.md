# Actividad 2: Desarrollo Web Colaborativo

## Nota sobre la modalidad de trabajo
Aunque la actividad fue planteada originalmente para ser realizada en equipo, he decidido llevarla a cabo de forma **individual**. Debido a mis responsabilidades laborales actuales, la coordinación de horarios con otros compañeros representaba una dificultad logística significativa.

Para cumplir con los objetivos  de la actividad, he asumido todos los roles (Líder y Colaboradores), simulando el flujo de trabajo grupal mediante la creación de ramas independientes para cada sección del sitio y la gestión de sus respectivos Pull Requests.
